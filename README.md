# 🔧 Crystal Beauty Clear – Backend

This repository contains the **backend (Node.js + Express)** for the Crystal Beauty Clear e-commerce system.  
It provides RESTful APIs for authentication, product management, cart handling, and order processing.

---

## 🚀 Features

- 🔐 Secure user authentication using JWT
- 🌐 Google OAuth integration for login
- 👥 Role-based access control (Admin / Customer)
- 📦 Product management (CRUD operations)
- 🛒 Cart and order management APIs
- 💳 Checkout processing with total, discount, and net total calculation
- 📡 RESTful API architecture

---

## 🛠️ Technologies Used

- Node.js  
- Express.js  
- MongoDB  
- Mongoose  
- JSON Web Token (JWT)  
- Google OAuth  
- dotenv
